Koga for iPhone

Support

Questions, beta feedback, and reproducible bug reports help make Koga better.

Contact Notavello

Use the Notavello contact page for Koga for iPhone support, release questions, or beta feedback.

Include with a report

  • iPhone model and iOS version
  • Koga version and whether it came from TestFlight or the App Store
  • The website address, when the issue concerns a particular page
  • Clear steps that reproduce the problem
  • What you expected and what happened instead
  • A screenshot or screen recording with private information removed

Before reporting a website problem

Try the same page again, confirm the phone has a working connection, and note whether the behavior also occurs in another iPhone browser. This helps separate a website issue from a Koga-specific issue.

Installation support

Koga’s normal iPhone distribution paths are the App Store and TestFlight. Do not install an IPA sent through an unofficial link. If a beta invitation fails, include the TestFlight error text in your report.
