Manual Android installation

This is an alternative installation method for users who cannot download Koga from Google Play. You may need to allow your browser to install unknown apps.

Download Koga APK

Only download Koga from this official Notavello page or the official Notavello GitHub repository.

Manual installation steps
  1. Download the Koga APK.
  2. Open the downloaded file.
  3. If prompted, allow your browser to install unknown apps.
  4. Return to the download and tap Install.
