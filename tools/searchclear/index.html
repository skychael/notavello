Designed for searchers who want control.

SearchClear does not replace Google. It adds a practical layer of controls to the results page and builds cleaner Google search URLs from your choices.

  • Works directly on Google search result pages.
  • Supports custom domains you want to block permanently.
  • Includes an optional context-menu search action for selected text.

Privacy-focused by default.

The extension stores your SearchClear settings locally through Chrome extension storage. It does not require a SearchClear account, does not send your settings to a SearchClear server, and does not include analytics in the extension.

  • No account required.
  • No extension analytics.
  • No sale or sharing of personal information by SearchClear.

Install & workflow

🧩

Install from the Chrome Web Store

Install SearchClear from the official listing when the Chrome Web Store review is approved. Before approval, the listing may not be publicly accessible yet.

Open Chrome Web Store listing →

1

Search Google normally

Open any Google results page. SearchClear places the toolbar above the results where you already work.

2

Choose filters

Click the pill controls you want. SearchClear updates the Google search URL with the right exclusions and search parameters.

3

Keep searching

Use sticky filters to remember your setup, or reset back to regular results whenever you need to.

Support for SearchClear

If something is not working, include your Chrome version, operating system, the Google search URL where it happened, and which filters were active. Avoid sending sensitive search terms in support messages.
