Usage Meter for Claude Privacy Policy

Effective date: July 5, 2026

Usage Meter for Claude is an unofficial browser extension for Chrome. It is not affiliated with, endorsed by, or sponsored by Anthropic.

What the extension does

The extension displays Claude.ai usage-limit information for the signed-in Claude account in the user's browser, including the 5-hour usage percentage, the 7-day usage percentage when available, and reset timing when Claude provides it.

Data collection

Usage Meter for Claude does not collect, sell, or share user data. The extension does not operate a Notavello server and does not transmit prompts, messages, answers, account details, browsing history, or page content to Notavello or to any third party.

Local storage

The extension uses Chrome local storage only to save the organization identifier needed to request Claude usage information and the last displayed usage percentage so the toolbar icon can show the most recent state. This data remains local to the user's browser.

Network access

The extension requests access to https://claude.ai/* so it can use the user's existing signed-in Claude session to request usage-limit information from Claude.ai. The extension does not read Claude conversations or page content.

Remote code

The extension does not use remote code. All JavaScript is packaged with the extension. Network requests are used only to retrieve usage-limit data from Claude.ai, not to load or execute remote scripts.

Contact

Questions about this privacy policy can be directed through Notavello's contact page at /pages/contact.
