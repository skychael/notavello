Designed to answer one question quickly.

When you are planning a long work session, the last thing you want is a surprise usage wall. Usage Meter for Claude keeps the key numbers close by: the current 5-hour window, the 7-day window, and reset timing if it is available.

  • Small toolbar popup with a familiar dark interface.
  • At-a-glance color changes as usage gets higher.
  • Works from the Chrome toolbar while you stay signed in at claude.ai.

Privacy-forward by default.

The extension does not run a Notavello server, does not sell data, and does not collect prompts, answers, or chat content. It only requests access to claude.ai so it can read the usage-limit information shown to your own account.

Unofficial companion

Usage Meter for Claude is not affiliated with, endorsed by, or sponsored by Anthropic. It may stop working if Claude changes the usage endpoint.

  • No prompts, chats, or message content collected.
  • No third-party analytics inside the extension.
  • Usage state is kept locally in Chrome storage.

Simple workflow

Stay signed in at claude.ai.The extension uses your normal browser session to request your own Claude usage-limit information.
Open the toolbar popup.See your 5-hour and 7-day usage status without digging through account pages or guessing where you stand.
Refresh when needed.Use the refresh control for a fresh reading, while the toolbar ring keeps the most recent state visible between checks.

Support for Usage Meter for Claude

If something is not working, include your Chrome version, operating system, whether you are signed in at claude.ai, and what the popup displayed. The extension does not collect diagnostics itself.
