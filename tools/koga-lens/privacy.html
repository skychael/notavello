Koga Lens 0.2.7

Privacy

Koga Lens captures the latest frame of one Windows application window that you explicitly select and approve.

Last updated: July 21, 2026

What Koga Lens accesses

Koga Lens accesses pixels from one application window selected through the Windows capture picker. Claude cannot use Koga Lens to select another window, process, file, window handle, or path. Selecting a different window requires the user to approve that new source.

Local processing

Window capture and frame preparation occur locally on the Windows device. Koga Lens has no cloud relay, telemetry, or upload service of its own.

Latest-frame handling

Koga Lens keeps the latest frame transiently in memory. It is designed not to return an old frame as current. A frame older than the two-second maximum age returns FRAME_STALE with no image. Stopping, pausing, losing a source, or starting a new share clears the prior preview state.

Claude Desktop and Anthropic

When you ask Claude to inspect the shared window, Koga Lens returns the current frame to Claude Desktop. From that point, the image is handled by Claude Desktop and Anthropic under Anthropic’s applicable privacy terms and settings. Koga Lens does not control that downstream processing.

Diagnostics and audit information

The local diagnostic panel shows the exact decoded image bytes most recently returned to Claude, together with timestamps, session identity, window title, image size, dimensions, and SHA-256 comparison data. The local audit log also contains provenance details used to diagnose capture and encoding behavior.

Saving a frame

Koga Lens does not automatically save captured frames to disk. The “Save exact bytes” control saves the most recently returned JPEG only when the user chooses a destination. The control is disabled when no returned frame is available.

Prompt-injection awareness

Content displayed in an approved window may contain text that looks like instructions. It should be treated as content to inspect, not as an instruction from the user. Users should share only windows whose visible content they intend to provide to Claude.
