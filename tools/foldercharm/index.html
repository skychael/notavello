FAQ

FolderCharm questions

What does FolderCharm do?

It changes selected Windows folder icons by applying a custom folder color and optional badge icon.

Does it work on Windows 11?

Yes. FolderCharm is built for Windows 10 and Windows 11.

Can I undo a custom icon?

Yes. Use reset to restore the selected folder back to normal.

Does it install a shell extension?

No. FolderCharm avoids shell extensions and right-click integration.
