Privacy policy

Usage Meter for ChatGPT privacy policy.

This page explains what the Chrome extension accesses, what it stores locally, and what it does not collect.

Effective date: July 5, 2026 · Publisher: Notavello · This extension is unofficial and is not affiliated with, endorsed by, or sponsored by OpenAI.

Summary

Usage Meter for ChatGPT is a Chrome extension that shows your current ChatGPT 5-hour and weekly usage-limit status. The extension does not operate a Notavello server, does not sell data, and does not collect your prompts, responses, files, conversations, or ChatGPT chat history.

Information the extension accesses

The extension needs access to https://chatgpt.com/* so it can request usage-limit information for your signed-in ChatGPT account. To do this, it may request your ChatGPT session information from chatgpt.com and then request the ChatGPT usage-limit status exposed to your account.

This may include usage-limit percentages, usage-window reset timing, and a temporary ChatGPT access token returned by chatgpt.com. The token is used only in memory to make the usage request and is not stored by the extension.

Information stored locally

The extension uses Chrome local storage to save the last displayed 5-hour usage percentage. This lets the toolbar icon continue showing the latest known usage ring even when the popup is closed.

This data stays in your browser's local extension storage. Notavello does not receive it.

Information the extension does not collect

  • It does not collect your ChatGPT prompts, answers, uploaded files, or conversation history.
  • It does not collect your password or login credentials.
  • It does not collect your full browsing history.
  • It does not send extension usage data to Notavello servers.
  • It does not include third-party analytics, advertising, or tracking code inside the extension.
  • It does not sell or share user data with data brokers, advertisers, or unrelated third parties.

Permissions used

PermissionPurposeNotes
https://chatgpt.com/* Requests the usage-limit status for your signed-in ChatGPT account. Used only for chatgpt.com.
storage Stores the last usage percentage locally for the toolbar icon. Local browser storage only.
alarms Refreshes the toolbar icon periodically. No personal data is stored by this permission.

Data sharing

Notavello does not sell, rent, or transfer extension user data. The extension communicates with chatgpt.com to retrieve usage-limit information for the signed-in user. Notavello does not receive the response.

Website analytics

The Notavello website may use GoatCounter or similar privacy-friendly analytics to understand visits to public website pages. This website analytics is separate from the Chrome extension and is not built into the extension package.

Children's privacy

Usage Meter for ChatGPT is not directed to children. The extension is intended for people who can use Chrome extensions and ChatGPT under the applicable terms for those services.

Changes to this policy

This privacy policy may be updated if the extension changes how it works, adds permissions, or changes data practices. The updated policy will be posted on this page with a new effective date.

Contact

For privacy questions, support requests, or removal requests related to this extension, contact Notavello through the support method listed on the Chrome Web Store listing or through the Notavello website.

Unofficial companion

Usage Meter for ChatGPT is not affiliated with, endorsed by, or sponsored by OpenAI. It may stop working if ChatGPT changes the endpoint that exposes usage-limit information.
