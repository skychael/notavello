Designed to answer one question quickly.

When you are planning a long work session, the last thing you want is a surprise usage wall. Usage Meter for ChatGPT keeps the key numbers close by: the current 5-hour window, the weekly window, and the reset timing if it is available.

  • Small toolbar popup with a familiar dark interface.
  • At-a-glance color changes as usage gets higher.
  • Works from the Chrome toolbar while you stay signed in at chatgpt.com.

Privacy-forward by default.

The extension does not run a Notavello server, does not sell data, and does not collect prompts, answers, or chat content. It only requests access to chatgpt.com so it can read the usage-limit information shown to your own account.

Unofficial companion

Usage Meter for ChatGPT is not affiliated with, endorsed by, or sponsored by OpenAI. It may stop working if ChatGPT changes the usage endpoint.

  • No prompts, chats, or message content collected.
  • No third-party analytics inside the extension.
  • Stores only the last usage percentage locally for the toolbar icon.

Simple workflow

Sign in at chatgpt.com. The extension reads usage-limit status from your existing ChatGPT session. It cannot work if you are signed out.
Click the toolbar icon. Open the popup to see your 5-hour and weekly usage status, plus reset estimates when available.
Use the ring as a quick signal. The toolbar icon shows the latest known short-window usage percentage so you can keep working without opening another tab.

Support for Usage Meter for ChatGPT

If something is not working, confirm that you are signed in at chatgpt.com, then include your Chrome version, operating system, and what the popup showed. Do not send private chat content.
